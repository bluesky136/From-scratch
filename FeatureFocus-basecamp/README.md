# Basecamp Demo

This app aims to replicate some of the features from Basecamp.

![](https://raw.githubusercontent.com/codeschool/FeatureFocus/basecamp/docs/basecamp.png)

* Timeline / Progress
* Infinite Scrolling
* Search
* Email Notifications

## Setup

* run `bundle`
* run `rake db:setup`
    * This will also load seed data under **db/seeds.rb**
